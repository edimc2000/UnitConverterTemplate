# UnitConversion (Java)

A simple Java console project that demonstrates unit conversion functionality.

Project files
- `UnitConversion.java`: Core conversion logic and helper methods.
- `UnitConversionDriver.java`: Example driver that demonstrates how to use the conversion utilities from the command line.
