# UnitConversion (Java)

A simple Java console project that demonstrates unit conversion functionality.

Project files
- `UnitConversion.java`: Core conversion logic and helper methods.
- `UnitConversionDriver.java`: Example driver that demonstrates how to use the conversion utilities from the command line.

Build

Open a terminal in the `scratches` directory and run:

```sh
javac UnitConversion.java UnitConversionDriver.java
```

Run

```sh
java UnitConversionDriver
```

If the driver expects command-line input, follow the prompts shown after running the program.

Example

Typical usage (interactive):

```
> java UnitConversionDriver
Enter value to convert: 100
Enter source unit (e.g., m, km, ft): m
Enter target unit (e.g., km, mi, ft): km
Result: 0.100 km
```
